This installation contains ITTO Explorer v1.0
To run ITTO Explorer on windows machine follow these steps:
	1.Extract the ITTO.zip into C:\ drive.This will create a folder "ITTO" in C:\
	2.Make sure you have a enviroment variable defined "JAVA_HOME" which points to JDK or JRE folder
	3.To check if JAVA_HOME is set, open a command prompt and type echo %JAVA_HOME%.If set it will disply the current java home directory.
	4.To define a JAVA_HOME variable on windows right click  "My Computer"
	5.Click "Properties" in "Advanced" tab 
	6.Click "Environment Variables".
	7.In "System Variables" click New
	8.Enter the variable name as JAVA_HOME.	
	9.Enter the variable value as the installation path for the Java Development Kit(JDK) or Java Runtime Environment(JRE)
	10. e.g. "C:\Program Files\Java\jdk1.7.0_21" or "C:\Program Files\Java\jre7"
	11.Double click run.bat and your are done.
